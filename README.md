# 🚀 Projet Fullstack Auth (Node.js & Vanilla JS)

Ce projet est une application web complète incluant un système d'authentification sécurisé avec **JWT (JSON Web Tokens)**, une base de données **MongoDB**, et un front-end en **JavaScript Vanilla**.

## 🛠️ Technologies utilisées

* **Backend :** Node.js, Express.js
* **Base de données :** MongoDB (via Mongoose)
* **Sécurité :** JWT pour l'authentification, bcrypt pour le hachage des mots de passe
* **Frontend :** HTML5, CSS3, JavaScript Vanilla (ES6+)

## 📋 Prérequis

Avant de commencer, assurez-vous d'avoir installé :
* [Node.js](https://nodejs.org/) (v18+)
* [MongoDB](https://www.mongodb.com/try/download/community) (ou un compte MongoDB Atlas)
* Un terminal configuré (Bash ou Zsh)
